⛔ **コードレビューの指摘事項が未修正です**

コードレビューで検出された問題を修正せずに終了しようとしています。

## 必須の対応

CLAUDE.md のルールに従い、**スコア 50 以上の指摘事項**に対して修正を完了してください：

1. 検出された問題をすべて確認
2. 各問題に対して修正を実施
3. 修正内容をコミット・プッシュ
4. PR を更新

## 確認事項

- [ ] スコア 50 以上の指摘事項をすべて修正したか？
- [ ] 修正内容をコミット・プッシュしたか？
- [ ] CI が成功しているか？

問題を修正してから、再度停止してください。

## 注意

このルールは現在 **無効化** されています。

理由: `hookify.code-review-immediate-fix.local.md` で `PostToolUse` イベントによりコードレビュー実行直後にブロックされるため、停止時の無条件ブロックは不要です。

停止時の安全網が必要な場合は、スコア判定を追加して条件化してください。
