# Shell Script Coding Rules

## Documentation

- Write function description comments. Language: follow the project CLAUDE.md if it specifies one; otherwise English

## Error Messages

- Write error messages in English

## Formatting

- Indent with 4 spaces

## Shebang

- Use `#!/bin/bash` (use `#!/bin/sh` only when POSIX sh is sufficient)
